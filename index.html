Git Workflow

Principais Comandos

Verificar estado dos arquivos/diretórios

git status

Adicionar alterações ao commit

git add "nome_do_arquivo" / git add . / git add --all

Remover arquivo/diretório

git rm "nome_do_arquivo" / git rm -r "nome_do_diretorio"

Realizar um commit

git commit / git commit -m "nome_do_commit"

Enviar alterações para o repositório remoto

git push

Atualizar os arquivos no branch atual

git pull

Buscar as alterações, mas não aplicá-las no branch atual

git fetch

Visualizar histórico de commits

git log / git log --oneline / git log --oneline --all / git log --oneline --all --graph

Navegar para uma versão específica

git checkout "versao_desejada" / git checkout HEAD~1

Criar e alternar para uma nova branch

git checkout -b "nome_da_nova_branch"

Voltar para o branch principal

git checkout main

Mesclar uma branch na branch principal

git merge "nome_da_branch"

Fluxo Padrão

git status
git add "nome_do_arquivo"

git commit -m "nome_do_commit"
